Calorie Counter
A functional, responsive web application built to help users track their daily caloric intake and expenditure. This project was developed as part of the freeCodeCamp JavaScript Algorithms and Data Structures curriculum.

## Features
Dynamic Entry Addition: Users can add multiple entries for Breakfast, Lunch, Dinner, Snacks, and Exercise.

Calorie Calculation: Automatically calculates the total calories consumed vs. calories burned.

Budget Tracking: Compares the net calories against a user-defined daily calorie budget to show a surplus or deficit.

Input Validation: Uses Regular Expressions (Regex) to clean user input and prevent invalid character entries (like +, -, or spaces in number fields).

Clear Functionality: Allows users to reset the form and start over with a single click.

## Technical Skills Applied
JavaScript DOM Manipulation: Using getElementById, querySelector, and template literals to dynamically update the UI.

Event Handling: Managing form submissions and button clicks.

Regular Expressions: Implementing RegExp to sanitize string inputs and ensure mathematical accuracy.

CSS Layouts: Using a clean, accessible design to ensure the counter is easy to read on all devices.

## How to Use
Enter your Daily Calorie Budget.

Select a category (e.g., Breakfast) and click "Add Food".

Enter the name and calorie amount for each item.

Click "Calculate" to see your remaining calories and overall tally.
